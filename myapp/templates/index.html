Full-Stack Developer - React - Python

Web application for managing routes of the integrated bus transportation system in the city of Valledupar, implementing a graph-oriented non-relational database.

As a full-stack developer and project lead, I have been working on an exciting web-based platform aimed at optimizing public transportation route planning. Leveraging Python with Flask in the backend, I established secure connections with the Aura DB (Neo4j) database. The backend utilizes a graph-based data model to manage complex relationships between transportation coordinates, enabling efficient and personalized route options. I execute optimized Cypher queries to extract real-time and accurate information on available routes. For the frontend, I utilized the powerful JavaScript library, React, to design a modern and intuitive user interface. The web application prioritizes responsiveness and usability, significantly enhancing the user experience during transportation journey planning. As the project progresses, my dedication remains focused on creating a comprehensive and functional solution that simplifies and optimizes users' daily commutes, improving urban mobility and delivering an exceptional user experience.
